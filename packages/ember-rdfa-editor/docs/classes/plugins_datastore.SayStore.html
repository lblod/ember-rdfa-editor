    • match(
          subject?: SubjectSpec,
          predicate?: PredicateSpec,
          object?: ObjectSpec,
      ): this

      Transformer method. Filters out any triples which do not match the given pattern. Pattern matching behaves similar as in SPARQL (but don't stretch that metaphor too far).

      Passing in undefined for any of the arguments means any term will match it.

      This method accepts concise term syntax, as defined in ConciseTerm
